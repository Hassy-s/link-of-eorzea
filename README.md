# Eorzea Link Hub V3

FF14関連サイトを、ジャンル・サブカテゴリごとに整理できる個人用リンク集です。  
GitHub Pagesでそのまま公開できます。

## ファイル

- `index.html`
- `style.css`
- `app.js`

## GitHub Pagesで使う方法

1. 新しいリポジトリを作成します。
2. このフォルダ内の3ファイルを、リポジトリの一番上にアップロードします。
3. GitHubの `Settings` → `Pages` を開きます。
4. `Deploy from a branch` を選びます。
5. Branchを `main`、フォルダを `/(root)` にして保存します。

## 主な機能

- ジャンルと1階層のサブカテゴリ
- サイト追加・編集・削除
- お気に入り
- 検索
- ドラッグ＆ドロップによる並び替え
- JSONバックアップと復元
- 画像表示「大・小・なし」
- カテゴリ名・画像サイズ・検索・サイト追加を横一列にまとめた省スペース表示
- YouTubeサムネイル自動取得
- faviconの小さい中央表示
- 画像URLの手動指定
- 編集画面での画像取得方法変更とプレビュー

## 画像の動き

サイト追加時は自動判定です。

1. YouTube URLならYouTubeサムネイル
2. それ以外はfavicon

保存後に管理モードから編集すると、次の方式へ変更できます。

- 自動
- 画像URL
- YouTube
- favicon

## データ保存

登録内容と表示設定はブラウザの `localStorage` に保存されます。  
別のPCやブラウザへ移すときは、管理モードの「バックアップ」と「復元」を使ってください。

## 以前のV3データについて

同じドメイン・同じブラウザで公開する場合、保存キーは以前と同じ `eorzeaLinkHubV3` を使用しているため、基本的には既存データを引き継ぎます。

新しいGitHub PagesのURLへ移動する場合、ブラウザ上では別サイトとして扱われるため、旧サイトでJSONバックアップを書き出してから、新サイトで復元してください。
